# Olympus

AI-powered development platform with multi-agent orchestration.

## Quick Install

```bash
npm i -g olympus-dev
```

## Basic Usage

### Authentication

```bash
# Authenticate with AI providers
olympus auth
```

### Run AI Agent Task

```bash
# Execute a development task with AI agents
olympus run "Add a login form to the homepage"
```

### Apply Code Patches

```bash
# Apply AI-generated code patches
olympus patch path/to/patch.diff
```

### List Available Agents

```bash
# View all registered AI agents
olympus agents
```
